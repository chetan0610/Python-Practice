# 📅 Day 09 - Functions

## 📖 Overview

This folder contains my Day 09 Python practice programs focused on **Functions**.

Functions are one of the most important concepts in Python. They help organize code, improve readability, reduce repetition, and make programs easier to maintain. This practice covers function creation, parameters, return values, recursion, lambda functions, variable scope, and real-world applications.

---

## 🎯 Learning Objectives

- Understand the purpose of functions
- Create and call functions
- Work with parameters and arguments
- Return values from functions
- Use default parameters
- Use keyword arguments
- Work with variable-length arguments (`*args` and `**kwargs`)
- Understand local and global scope
- Use lambda functions
- Implement recursion
- Build real-world programs using functions
