# 📅 Day 10 - File Handling

## 📖 Overview

This folder contains my Day 10 Python practice programs focused on **File Handling**.

File handling is an essential Python skill used to create, read, write, update, and manage files. It is widely used in automation, data processing, logging, report generation, and real-world software development.

---

## 🎯 Learning Objectives

* Understand file handling in Python
* Create and write text files
* Read data from files
* Append data to existing files
* Read files line by line
* Count lines, words, and characters
* Search and replace text in files
* Copy and merge files
* Delete files safely
* Check if a file exists
* Work with CSV files
* Build practical file-based applications
