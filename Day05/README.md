# 📘 Day 05 – Lists & Problem Solving

## 📖 Overview

Day 05 focuses on mastering Python lists through practical problem-solving. These programs strengthen understanding of list manipulation, searching, comparison, rotation, and algorithmic thinking.

---

## 📂 Programs

### 1. LargestNumber.py
Find the largest number in a list without using `max()`.

### 2. SmallestNumber.py
Find the smallest number in a list without using `min()`.

### 3. SecondLargestNumber.py
Find the second largest distinct number in a list using loops and comparisons.

### 4. RemoveDuplicatesFromList.py
Remove duplicate elements while preserving the original order.

### 5. MergeLists.py
Merge two lists without using `+` or `extend()`.

### 6. CommonElements.py
Find common elements between two lists without using `set()`.

### 7. RotateListLeft.py
Rotate a list one position to the left using manual shifting.

### 8. RotateListRight.py
Rotate a list one position to the right using manual shifting.

### 9. ReverseList.py
Reverse a list using the two-pointer technique without using built-in reverse methods.

### 10. IsListSorted.py
Check whether a list is sorted in ascending order.

---

## 🧠 Concepts Covered

- List creation and traversal
- User input and type conversion
- Searching algorithms
- Manual maximum and minimum finding
- Duplicate removal
- List merging
- List rotation
- Two-pointer technique
- Comparison-based algorithms
- Problem-solving using loops and conditions
